# 17 - Roadmap after Phase 1

## Phase 2 - Xuất kho / sử dụng vật tư

Goal:

```text
Ghi nhận vật tư được sử dụng cho từng ao.
```

Features:

```text
- Tạo phiếu xuất kho
- Chọn ao
- Chọn vật tư
- Số lượng sử dụng
- Mục đích sử dụng
- Tồn kho giảm sau khi duyệt
```

## Phase 3 - Quản lý ao

Features:

```text
- Tạo ao
- Diện tích
- Ngày thả giống
- Mật độ thả
- Loài tôm
- Trạng thái ao
```

## Phase 4 - Theo dõi môi trường nước

Metrics:

```text
pH
DO
Nhiệt độ
Độ mặn
Kiềm
NH3
NO2
```

## Phase 5 - Theo dõi tăng trưởng

Features:

```text
- Cân tôm
- Trọng lượng trung bình
- Số con/kg
- Tỷ lệ sống ước tính
- FCR
```

## Phase 6 - Báo cáo chi phí

Reports:

```text
- Chi phí vật tư
- Chi phí theo ao
- Chi phí theo vụ
- Tồn kho hiện tại
- Giá trị tồn kho
```

## Phase 7 - Backup/export

Features:

```text
- Export Excel
- Backup database
- Restore database
- Report PDF
```

## Phase 8 - Optional IoT

Only after core app is stable.

Features:

```text
- Kết nối thiết bị đo nước
- Gửi dữ liệu tự động
- Cảnh báo chỉ số bất thường
```

## Phase 9 - AI / prediction

Only after enough data.

Possible features:

```text
- Dự báo rủi ro môi trường
- Gợi ý xử lý nước
- Dự báo chi phí
- Dự báo tăng trưởng
```
